## Usage

> TrustPool allows friends and acquaintances to come together and pool money to build up their purchasing power.

## Requirements

- Node 10.5.x
- Postgresql 10
- Angular 6

## Development

### Installing Dependencies and Startup

From within the root directory:

```sh
cd trust-pool-app
npm install
npm run build
npm start
```
